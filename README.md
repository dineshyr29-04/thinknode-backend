# ThinkNode Backend
This is the backend repository for the ThinkNode platform, supporting real-time updates, customer order management, and secure admin authentication.

## Stack
- Node.js
- Express
- Socket.IO
- MySQL
- JWT Auth
- Multer (File Uploads)

## Setup
1. Run `npm install`
2. Configure `.env` based on `.env.example`
3. Run `npm run dev`
